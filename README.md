# varie-cli

A simple CLI for building Varie projects

```bash
$ npm install -g varie-cli
```

### Usage

```bash
$ varie new <project-name>
```

Example:

```bash
$ varie new my-project
```

The above command generates a brand new Varie project and is ready for development.
